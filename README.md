# Player Evaluation Model

## Overview
This repository contains all code and data needed to reproduce the analyses presented in the thesis "Developing a Comprehensive Player Performance Metric Using Expected Goals and Match Data to Enhance Scouting and Identify Undervalued Football Players". 
It includes scripts for data preprocessing, modeling, evaluation, visualization, and the resulting datasets.

## Repository Structure
- `code/` — Python scripts for preprocessing, model building, evaluation, and figures.  
- `results/` — CSV files containing model outputs, time series profiles, residuals, forecast comparisons, and other analysis results.  
- `README.md` — This file, explaining repository contents.
